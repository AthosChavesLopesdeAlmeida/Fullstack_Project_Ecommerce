# Fullstack_Project_Ecommerce

**Paleta sugerida:**

| Papel | Cor | Hex |
|---|---|---|
| Fundo principal | Off-white / marfim | `#F5F1E8` |
| Texto principal | Marrom quase preto | `#2B211A` |
| Primária (madeira escura) | Marrom café | `#5C4033` |
| Secundária (madeira média) | Marrom âmbar | `#8B5E3C` |
| Destaque/CTA | Dourado envelhecido | `#B08D57` |
| Borda/divisor | Bege acinzentado | `#D9CBB8` |
| Erro/alerta | Vermelho vinho | `#7A2E2E` |
| Sucesso | Verde oliva escuro | `#5A6B3C` |

**Lógica:**
- O marrom café (`#5C4033`) como cor primária remete diretamente à moldura de madeira, e funciona bem em botões/headers.
- O dourado envelhecido (`#B08D57`) dá um toque de "arte/quadro" sem parecer infantil — bom para CTAs (comprar, adicionar ao carrinho).
- Fundo em off-white (`#F5F1E8`) em vez de branco puro deixa mais aconchegante e menos "clínico", combinando com a seriedade pedida sem ficar frio.
- Vermelho vinho e verde oliva (em vez de vermelho/verde puros) mantêm a paleta coesa nos estados de erro/sucesso.
